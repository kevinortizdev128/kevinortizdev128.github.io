EduWeb - Sitio Educativo

EduWeb es un sitio educativo estático desarrollado con HTML, CSS y JavaScript.
El proyecto incluye secciones de cursos, información educativa y un formulario de contacto.
Está diseñado con buenas prácticas de desarrollo web, incluyendo diseño responsivo y una interfaz de usuario intuitiva para facilitar el aprendizaje en línea.

Características principales:
  • . Diseño responsivo y moderno
  • . Sección de cursos organizados
  • . Formulario de contacto funcional
  • . Navegación intuitiva
  • . Optimizado para diferentes dispositivos

Tecnologías utilizadas: HTML, CSS, JavaScript
